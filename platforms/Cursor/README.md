# Cursor — use as a skill

Cursor supports Agent Skills (`SKILL.md` with name and description). The LinkedIn profile audit skill is ready to load.

## Install (recommended)

1. Copy this repo folder into your Cursor skills directory, for example:
   - **Mac/Linux:** `~/.cursor/skills/linkedin-profile-audit/`
2. Ensure the folder contains:
   - `SKILL.md`
   - `references/`
3. Restart or reload Cursor.

## How to use

- In chat, invoke `/linkedin-profile-audit` (or ask to use the LinkedIn profile audit skill).
- The assistant will run onboarding first, then one-question-at-a-time collection, then scoring and rewrites.

## If you don't use a skills folder

- Paste `SKILL.md` into chat and ask the assistant to follow it.
