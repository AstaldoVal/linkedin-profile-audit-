# ClawdBot — workaround

ClawdBot skills are code-based and may not support direct markdown skill upload.

## Option 1: custom instructions (if supported)

1. Paste `Custom_Instructions.md` from this folder into ClawdBot custom instructions/system prompt.
2. Start chat and request the LinkedIn profile audit flow.

## Option 2: paste in chat

1. Start a new chat.
2. Paste `SKILL.md` and ask the assistant to follow it exactly.
3. Provide profile assets in the required order.
