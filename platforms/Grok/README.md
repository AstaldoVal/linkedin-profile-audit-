# Grok — custom instructions (no skill upload)

Grok supports custom instructions (Customize Grok / Workspace instructions). It does not support direct markdown skill upload.

## Use as "skill" (recommended)

1. Open **Customize Grok**.
2. In instructions, paste `Custom_Instructions.md` from this folder.
3. Save and start chat.

## If you don't use custom instructions

- Paste `SKILL.md` in the first message and ask Grok to follow it.
- Or use the one-shot prompt from the main README.
