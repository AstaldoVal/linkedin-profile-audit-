# Manus — use as a skill

Manus supports Skills (file-based upload including `SKILL.md`).

## Install

1. Open **Skills** in Manus.
2. Upload this repo folder (or a `.zip` of it).
3. If only a single file is accepted, upload `SKILL.md`.
4. Start chat and invoke the skill.

## How to use

- Ask for `/linkedin-profile-audit`.
- The skill runs onboarding first, then section-by-section audit flow.

## If upload is not available

- Paste `SKILL.md` in chat and ask Manus to follow it exactly.
