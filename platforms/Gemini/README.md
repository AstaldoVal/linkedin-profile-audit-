# Gemini — use as a Gem

Gemini does not use markdown skill folders directly. The equivalent is a Gem with fixed instructions.

## Install

1. Open Gemini and create/edit a **Gem**.
2. Paste `Custom_Instructions.md` from this folder into Gem instructions.
3. Optionally upload both reference files from `references/`.
4. Save and use this Gem for LinkedIn profile audits.

## If you don't use Gems

- Paste `SKILL.md` into chat and ask Gemini to follow it.
