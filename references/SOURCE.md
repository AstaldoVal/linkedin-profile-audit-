2. **Profile Audit Examples.pdf** — niche walkthroughs (founders, coaches, PM, tech, etc.). Full text: `profile-audit-examples-extracted.txt`.
3. **Your Claude X LinkedIn Profile Auditor.docx** — Project setup on Claude.ai, disclaimer, voice/tone guidance. Full text: `claude-x-linkedin-auditor-docx-extracted.txt`.

When the playbook changes, edit `../SKILL.md` and replace `profile-audit-examples-extracted.txt` from a new PDF export if needed.
